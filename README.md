## 🚀 網站維護手冊 (SOP)

### 📝 如何新增「內容專區」文章
1. **上傳文章檔**：
   - 進入 `posts/` 資料夾。
   - 點擊 `Add file` > `Create new file`。
   - 檔名必須是 `英文.md` (例如 `trip-2026.md`)。
   - 內容使用 Markdown 語法撰寫。
2. **更新選單**：
   - 回到根目錄打開 `index.html`。
   - 搜尋 `const postList`。
   - 在陣列中新增一行：`{ file: '你的檔名.md', title: '顯示的標題' }`。
   - **注意**：前一行的末端必須有逗號 `,`。

### 📊 如何更新「資助進度條」
- 直接打開你的 **Google Sheet** (雲端試算表)。
- 修改「進度」百分比數字。
- 儲存後，網頁會在 5-10 分鐘內自動同步（由 Google 發布延遲決定）。
